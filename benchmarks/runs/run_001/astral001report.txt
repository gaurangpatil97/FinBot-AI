### TABLE 5 — Methodology:

* **Strict Grading Criteria**: To receive a "Correct" score under strict grading, the actual response must contain all key numeric facts, entity names, and conclusions specified in the expected answer. Numeric values must fall within a $\pm2\%$ tolerance, and if the data is expected to be reported as missing/blank, the actual answer must state this instead of generating or hallucinating values.
* **Lenient Grading Criteria**: An actual response is marked as "Correct" or "Partially Correct" if it gets the general direction right, identifies correct entities, and captures the core point of the expected answer. Partial points are awarded if some of the secondary metrics are missing, slightly out of tolerance, or if some years in a multi-year comparison are omitted but the provided ones are correct.
